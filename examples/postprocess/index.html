Post Process example with osgjs

This sample demonstrate how to do post process with a pre render of the scene and then use it with a post process shader.

Vignetting

Vignetting is add some darker color on corner of the image. You can find more explanation on wikipedia . There is also an implementation on geek3D . Play with the slider to test it.

Change value of x [0-1] 0.77
Change value of y [0-1] 0.42

Pixelation

Pixelate the scene. The shader comes from Geek3D.

Change value of x [0-300] 0.77
Change value of y [0-300] 0.77

Monitor
